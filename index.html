Se trata de una plantilla de texto modelo de varias páginas que puede copiar para construir su primera página jQuery Mobile. Esta plantilla contiene varios contenedores de "página" en el interior, a diferencia de una plantilla de página única que tiene sólo una página dentro de ella.

Sólo ver la fuente y copiar el código para empezar. Toda la CSS y JS está vinculada a las versiones de jQuery CDN así que esto es muy fácil de configurar. Recuerde que debe incluir una etiqueta meta ventana gráfica en la cabeza para ajustar el nivel de zoom.

Que enlace a las páginas internas haciendo referencia a la id de la página que desea mostrar. Por ejemplo, para enlace a la página con una id de "two", mi enlace tendría una href="#dos" en el código.
